# ✈️ Time-Pilot

Time-Pilot is a productivity dashboard that connects **time tracking**, **goal-oriented execution**, and **retrospective analysis** to prevent blind busywork.

## Tech Stack

### Frontend

- Next.js
- React + TypeScript
- shadcn/ui, Tailwind CSS
- Redux Toolkit, React Query
- Recharts, Chart.js

### Backend

- Python, FastAPI
- SQLAlchemy + Alembic
- PostgreSQL
- JWT-based Authentication

### Infra

- AWS (EC2, RDS, S3)
- Vercel (Frontend)
- Docker (Local Development)
- GitHub Actions (CI)
